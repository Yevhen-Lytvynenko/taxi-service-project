Зображення для лендингу покладіть у цю папку (корінь сайту: /landing/...).

Рекомендовані файли (опційно):

- hero.jpg — повноширинний банер за текстом героя (темний градієнт лишається зверху для читабельності).
- hero-side.jpg — колаж або фото праворуч у герої.
- service-city.jpg, service-airport.jpg, service-delivery.jpg, service-business.jpg — картки «Послуги».
- strip-promo.jpg — широкий промо-блок на жовтому тлі.
- how-it-works.jpg — ілюстрація кроків.
- apps-mockup.jpg — мокап додатку / скріншоти.
- contact-map.jpg — карта або фото офісу.

Формати: JPG або WebP, помірна вага для швидкого завантаження.

Якщо файлу немає, на сторінці показується підпис-плейсхолдер.
